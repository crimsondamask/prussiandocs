PrussianStudio

A Lightweight And Modern SCADA Software

Get Started In Less Than 5min ⏱️

Easy to Use

PrussianStudio was designed from the start to be easy to setup and use. It requires no installation, and the whole software including documentation should take no more than 50mb of disk space.

Modern

PrussianStudio offers you a modern interface that is designed to be intuitive. It also offers you a Human Machine Interface powered by the Godot game engine and capable of handling complex visualizations.

Powered by Rust

Rust is a modern fast and safe language that compiles to WASM which allows us to build fast applications. PrussianStudio harnesses the speed and reliability provided by these technologies.
